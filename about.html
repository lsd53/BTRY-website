About this website

This website allows you to query proteins in S.cevervisiae, commonly known as yeast, and find all interactions for them supported by publications. You can query any protein in yeast using one of three identifiers, Entrez Gene ID, Gene name and ORF name. The website will display a page with a table of all the interactions for that queried protein. The table includes all three identifiers for each protein, a link to the NCBI website for more information and all the publications that support this interaction with both pmid and psi-mi numbers that you can use to search for on the PubMed database. You can click on each gene entrez id and this will redirect you to the NCBI website. These features were implemented using a cgi script that queries two tables in a database containing all the information for interactions in the yeast. The styling of the website was done using pure css, a publicly available css module. The styling uses a color pallete called ocean tide. This css styling allows for the webpage to be easily resizeable and maintian its appearance accross multipledifferent platforms.
